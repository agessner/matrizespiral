# matrizespiral

Resolução do problema http://dojopuzzles.com/problemas/exibe/matriz-espiral/

## Descrição

Dado um número de colunas e um número de linhas, retornar uma matriz em espiral de fora para dentro no sentido horário.

## Exemplos

Entrada: 3 4

Saída:

|  |  |  |
|--|--|--|
| 1| 2| 3|
|10|11| 4|
| 9|12| 5|
| 8| 7| 6|
 

Entrada: 5 6

Saída:


|  |  |  |  |  |
|--|--|--|--|--|
|1 |2 |3 |4 | 5|
|18|19|20|21| 6|
|17|28|29|22| 7|
|16|27|30|23| 8|
|15|26|25|24| 9|
|14|13|12|11|10|
